# Montreval Advocacia

## Sobre o Projeto

Site institucional desenvolvido para o escritório Montreval Advocacia, apresentando serviços jurídicos de alta performance com design sofisticado e elegante.

**Autor**: Lucas Kikuthi

## Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion

## Instalação

```sh
# Clone o repositório
git clone <YOUR_GIT_URL>

# Navegue até o diretório
cd montreval-advocacia

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

## Scripts Disponíveis

- `npm run dev` - Inicia o servidor de desenvolvimento
- `npm run build` - Cria a build de produção
- `npm run preview` - Preview da build de produção
- `npm run lint` - Executa o linter
- `npm run test` - Executa os testes

## Estrutura do Projeto

```
src/
├── assets/          # Imagens e recursos
├── components/      # Componentes React
├── hooks/           # Custom hooks
├── lib/             # Utilitários
├── pages/           # Páginas da aplicação
└── test/            # Testes
```

## Deploy

Para fazer o deploy, execute:

```sh
npm run build
```

Os arquivos otimizados estarão na pasta `dist/`.
